# 🎯 InterviewIQ – Interview Questions Predictor Using Machine Learning

An AI-powered **Interview Questions Predictor** developed using **Python, Flask, Machine Learning, NLP, and Groq AI** to help candidates prepare for interviews by analyzing resumes, predicting job roles, extracting skills, and generating personalized interview questions.

---

## 📌 Project Overview

InterviewIQ is an intelligent web application designed to simplify interview preparation through automation and personalization. The system analyzes a candidate's resume to identify key skills and predicts the most suitable job role using Machine Learning. It then generates personalized interview questions with the help of Large Language Models (LLMs), enabling candidates to practice questions that match their profile and career goals.

The application combines **Machine Learning**, **Natural Language Processing (NLP)**, and **Generative AI** to provide an interactive and user-friendly interview preparation platform.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/sheemashaik17/InterviewIQ.git
```

### Open the Project Folder

```bash
cd InterviewIQ
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
