# Interview Questions Predictor Using Machine Learning

## 📌 Project Overview

The Interview Questions Predictor is a Machine Learning-based web application that predicts relevant interview questions based on a candidate's skills and selected job role. The system helps students and job seekers prepare for technical interviews by providing personalized interview questions through an easy-to-use web interface.

---

## ✨ Features

- 👤 User Registration and Login
- 📄 Resume Upload
- 💼 Job Role Prediction
- 🤖 Machine Learning-based Interview Question Prediction
- 📊 Confidence Score Display
- 📝 Session History
- 🎨 Responsive User Interface

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/sheemashaik17/interview-questions-predictor.git
```

### Open the Project

```bash
cd interviewiq-ml
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```
